# Projector

Read model builder. Events in, external output (DB, API, files). Creates query-optimized views from event streams.

**Abbreviation:** `prj`

Projectors accept events from a single domain and output to external systems, databases, event streams, or files. They build read-optimized projections that support efficient queries.
